# Netflix EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to discover trends, patterns, and insights about Netflix content.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Steps

### 1. Data Loading
Loaded Netflix dataset using Pandas.

### 2. Data Cleaning
- Handled missing values
- Converted date formats
- Created new features
- Removed inconsistencies

### 3. Exploratory Data Analysis
Answered:
1. Movies vs TV Shows
2. Top Producing Countries
3. Titles Added Per Year
4. Content Ratings
5. Top Genres and Duration Analysis

### 4. Visualizations
- Bar Chart
- Line Chart
- Histogram
- Pie Chart
- Scatter Plot
- Heatmap

### Key Insights
- Movies dominate Netflix content.
- United States is the largest content producer.
- Netflix content grew rapidly after 2015.
- TV-MA is the most common rating.
- Drama is the most popular genre.
